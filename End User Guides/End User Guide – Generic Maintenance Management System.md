# End User Guide
## Generic Maintenance Management System (CMMS)

---

## Overview of the System
The Generic Maintenance Management System (CMMS) helps organizations:
- Track maintenance requests
- Manage work orders
- Schedule preventive maintenance
- Monitor asset history

The system centralizes maintenance activities and improves operational efficiency.

---

## User Roles

### Requester
- Submits maintenance requests
- Tracks request status

### Technician
- Receives assigned work orders
- Updates work progress
- Completes maintenance tasks

### Maintenance Manager
- Reviews and assigns work orders
- Monitors maintenance performance
- Generates reports

---

## Logging In to the System

*Figure 1: CMMS login page*

1. Open the CMMS application.
2. Enter your **username** and **password**.
3. Select **Sign In**.

> Note: Contact your system administrator if you cannot access your account.

---

## Creating a Maintenance Request

*Figure 2: Create Maintenance Request screen*

Use this procedure when an asset requires repair or service.

1. Select **Create Request**.
2. Enter the required information:
   - **Asset Name**
   - **Issue Description**
   - **Priority**
   - **Location**
3. Select **Submit**.

After submission, the system generates a request ID for tracking.

---

## Viewing Request Status

*Figure 3: Request status tracking*

1. Go to **My Requests**.
2. Select a request from the list.
3. Review the current **status** and **assigned technician**.

Common statuses include:
- Open
- In Progress
- On Hold
- Completed

---

## Work Order Lifecycle

A maintenance request follows this lifecycle:

1. Request submitted
2. Work order created
3. Technician assigned
4. Work in progress
5. Work completed
6. Work order closed

Each status update is logged for audit and tracking purposes.

---

## Performing Maintenance Tasks (Technician)

*Figure 4: Technician work order view*

1. Open the assigned work order.
2. Review task details and instructions.
3. Update **Work Notes** as needed.
4. Change the status to **Completed** when finished.

> Tip: Add clear notes to help managers review completed work.

---

## Scheduling Preventive Maintenance

*Figure 5: Preventive maintenance scheduling*

Preventive maintenance helps reduce equipment failures.

1. Go to **Maintenance Schedules**.
2. Select **Create Schedule**.
3. Define the frequency and asset.
4. Save the schedule.

The system automatically generates work orders based on the schedule.

---

## Closing a Work Order

*Figure 6: Closing a work order*

Maintenance managers should verify work before closure.

1. Open the completed work order.
2. Review technician notes.
3. Confirm task completion.
4. Select **Close Work Order**.

---

## Viewing Maintenance History

1. Open the **Assets** module.
2. Select an asset.
3. Review maintenance history and previous work orders.

This data helps identify recurring issues.
